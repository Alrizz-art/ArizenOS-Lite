# ArizenOS Lite — Command Palette Guide

> Linux-inspired quick command launcher — built into ArizenOS Lite v1.1

---

## How to Open

- Tap **> CMD** button in the top status bar on the home screen
- Or tap **> CMD** in Workspace Mode
- Or navigate: Labs → (planned shortcut)

---

## Built-in Commands

| Command | Action |
|---------|--------|
| `>ai` | Open Arizen AI assistant |
| `>labs` | Open ArizenLabs hub |
| `>settings` | Open Arizen Settings |
| `>routines` | Open Automation Routines |
| `>monitor` | Open System Monitor |
| `>workspace` | Enter Workspace Mode |
| `>ram` | Boost RAM immediately |
| `>sysinfo` | Show system information |
| `>battery` | Show battery detail |
| `>wifi on` | Enable WiFi |
| `>wifi off` | Disable WiFi |
| `>bt on` | Enable Bluetooth |
| `>bt off` | Disable Bluetooth |
| `>calc` | Open calculator |
| `>note` | Quick note |
| `>alarm` | Set alarm (HH:MM) |

---

## Search Apps

Type any app name (without `>`) to search installed apps:

```
chrome        → opens Chrome
termux        → opens Termux
github        → opens GitHub
```

---

## AI Quick Ask

Type any question and press **Enter** — it goes directly to Arizen AI:

```
bagaimana cara reverse string di Python?
apa itu docker compose?
terjemahkan: the quick brown fox
```

---

## Tips

- Commands are **case-insensitive**
- Type `>` alone to see all system commands
- Type app name for app search
- **Press Back** or tap ✕ to close without executing

---

## Design Philosophy

Inspired by VS Code Command Palette (`Ctrl+Shift+P`) and macOS Spotlight — a unified, keyboard-friendly entry point for everything on the system.

On ArizenOS, it replaces the need to navigate multiple menus for common system tasks.
